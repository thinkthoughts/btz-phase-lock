\begin{abstract}
We study constraint-first reconstruction using a shared multi-slice neural surrogate trained on multiple probe observables. Prior results show that value matching alone leaves residual ambiguity: distinct geometries can fit the same observables. We introduce derivative-sensitive constraints to enforce local structure and perform a systematic probe sweep over subsets of three observables—EE, WL, and GEO.

In a synthetic held-out setting, we find that the minimal identifying set is EE $\oplus$ GEO $\oplus$ derivative. This configuration achieves the lowest held-out metric error, while the full probe stack is not minimal and WL is not required for optimal reconstruction. The results show that derivative information is necessary for structure, and that identifying sets can be strictly smaller than the available probe family.
\end{abstract}

\section{Introduction}

A recurring problem in reconstruction is the gap between value fit and structure fit. A model may reproduce observables while failing to uniquely recover geometry.

Project progression:
\begin{itemize}
\item v17: indistinguishable observables
\item v18: multi-slice reduces ambiguity
\item v19: derivative enforces structure
\item v20: probe sweep identifies minimal set
\end{itemize}
